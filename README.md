# Olympic Games Analysis Dashboard (1896–2020)

## Overview
This project presents an interactive Power BI dashboard analyzing over a century of Olympic Games data.  
It explores country-wise performance, medal distributions, and participation trends across both Summer and Winter Olympics.

The objective is to extract meaningful insights and visualize global Olympic performance patterns using data analytics techniques.

---

## Team Collaboration
This project was developed as a collaborative effort, where team members contributed to data preparation, analysis, visualization design, and insight generation.  
The teamwork ensured a well-structured and insightful dashboard with clear storytelling through data.

---

## Key Insights
- The United States leads the all-time Olympic medal count by a significant margin  
- Norway demonstrates strong dominance in Winter Olympic events  
- Countries with higher participation levels generally achieve better medal outcomes  
- Distinct performance patterns exist between Summer and Winter Olympic Games  

---

## Dashboard Features
- Total medal comparison across countries  
- Summer vs Winter Olympics performance analysis  
- Distribution of Gold, Silver, and Bronze medals  
- Country-wise ranking and comparison  
- Participation vs performance insights  
- Interactive filters for deeper exploration  

---

## Tools Used
- Microsoft Power BI  
- Microsoft Excel

## Data Source
The dataset used in this project was collected from publicly available Olympic records:

- Wikipedia: All-time Olympic Games medal table  
https://en.wikipedia.org/wiki/All-time_Olympic_Games_medal_table  

The data was cleaned, structured, and transformed using Excel before being imported into Power BI for visualization.
